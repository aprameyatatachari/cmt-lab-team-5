# Order Management System - UI

Modern Angular frontend for the Order Management System.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:4200](http://localhost:4200) in your browser.

## Backend Connection

The UI connects to the WebSocket server at `ws://localhost:8080`. Ensure the Order Service backend is running before starting the UI.

## Features

- Real-time order updates via WebSocket
- Modern, minimalistic UI design
- Live connection status indicator
- Animated order entries
- Buy/Sell color-coded badges
- Responsive design for mobile and desktop
